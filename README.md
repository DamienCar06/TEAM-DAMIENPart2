# TEAM-DAMIENPart2

This is a Chess Game GUI built with Java Swing. It displays an 8x8 chessboard with alternating light and dark squares, initial piece positions for both players, and allows players to move pieces by clicking or dragging. Capturing opponent's pieces removes them from the board, and capturing the King ends the game with a winner announcement.

Features:
- Includes a game panel that keeps track of piece moves, captured pieces, and ability to reset game or undo move
- Includes a menu bar on the top left of the board that allows you to create a new game, save game, and load game
